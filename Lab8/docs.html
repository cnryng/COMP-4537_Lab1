Get Pokemons

Returns json data about multiple pokemon

  • URL

    /api/v1/pokemons?count=after=

  • Method:

    GET

  • URL Params

    Optional:

    • count=[integer]
    • after=[integer]

  • Data Params

    None

  • Success Response:

    Content: One or more pokemon documents

  • Error Response:

    Content: { msg : "after and count query parameters must be numbers" }

    OR

    Content: { msg: <mongoDB query error> }

Get a Pokemon

Returns json data about a single pokemon by id

  • URL

    /api/v1/pokemons/:id

  • Method:

    GET

  • URL Params

    Required:

    • id=[integer]

  • Data Params

    None

  • Success Response:

    Content: One pokemon document

  • Error Response:

    Content: {msg:Pokemon with id ${req.params.id} does not exists}

Create a Pokemon

Inserts a pokemon document

  • URL

    /api/v1/pokemon/

  • Method:

    POST

  • URL Params

    None

  • Data Params

    {

      "id": [integer],
  "name": {
      "english": [string],
      "japanese": [string],
      "chinese": [string],
      "french": [string]
  },
  "type": [string array],
  "base": {
      "HP": [integer],
      "Attack": [integer],
      "Defense": [integer],
      "Sp Attack": [integer],
      "Sp Defense": [integer],
      "Speed": [integer]
  }

    }

  • Success Response:

    Content: {msg: "Successfully inserted new pokemon"}

  • Error Response:

    Content: {msg: <mongoDB insertion error>}

Upsert a Pokemon

Updates a pokemon given an id; inserts new Pokemon if id does not exist

  • URL

    /api/v1/pokemon/:id

  • Method:

    PUT

  • URL Params

    Required:

    • id=[integer]

  • Data Params

    {

      "id": [integer],
  "name": {
      "english": [string],
      "japanese": [string],
      "chinese": [string],
      "french": [string]
  },
  "type": [string array],
  "base": {
      "HP": [integer],
      "Attack": [integer],
      "Defense": [integer],
      "Sp Attack": [integer],
      "Sp Defense": [integer],
      "Speed": [integer]
  }

    }

  • Success Response:

    Content: { msg : "Couldn't find pokemon with id ${req.params.id}; inserted new pokemon instead" }

    OR

    Content: { msg: "Successfully updated pokemon with id ${req.params.id}" }

  • Error Response:

    Content: {msg: <mongoDB insertion or insertion error>}

Patch a Pokemon

Updates a pokemon given an id; data in body can be an incomplete document.

  • URL

    /api/v1/pokemon/:id

  • Method:

    PATCH

  • URL Params

    Required:

    • id=[integer]

  • Data Params

    {

      "id": [integer],
  "name": {
      "english": [string],
      "japanese": [string],
      "chinese": [string],
      "french": [string]
  },
  "type": [string array],
  "base": {
      "HP": [integer],
      "Attack": [integer],
      "Defense": [integer],
      "Sp Attack": [integer],
      "Sp Defense": [integer],
      "Speed": [integer]
  }

    }

  • Success Response:

    Content: {msg:Successfully patched pokemon with id ${req.params.id}}

  • Error Response:

    Content: {msg:Patch failed. Pokemon with id ${req.params.id} does not exist}

  • Method:

    DELETE

  • URL Params

    Required:

    • id=[integer]

  • Data Params

    • None

  • Success Response:
